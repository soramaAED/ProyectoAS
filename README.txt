Para importar los datos correctamente se debe crear una carpeta 
llamada "data" dentro del directorio del proyecto. Se deberá de incluir 
las tres carpetas en las que se divide el dataset (soleado, nublado, noche)
dentro de "data" y que se puede encontrar compartidas en la carpeta
https://drive.google.com/drive/folders/1Y4RSSCIy-0t54gqVjyfz75-CjhNlWDEj
de Google drive.
La estructura resultado deberá tener la siguiente forma:


.Proyecto.rmd
.Memoria.docx
.Memoria.pdf
.Data ----------> /Data .soleado
			.nublado
			.noche
.parametros_modelos.R
.generacionMetadatos.R
